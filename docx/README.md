## 機能設計 詳細（P0〜P3）

### ユーザー/認証
- P0: ログイン / ログアウト / 初回ログイン（仮パスワード発行）
- P1: 管理者アカウント登録 / アカウント停止・無効化
- P2: パスワード再設定 / メールアドレス本人確認
- P3: 権限カスタム（部署単位） / 外部ID連携

### マイページ
- P0: 自分の投稿一覧 / 自分の投稿へのコメント一覧
- P1: プロフィール編集 / 記事の閲覧履歴
- P2: いいね記事一覧 / 公開範囲設定
- P3: 興味情報の設定

### トップページ
- P0: 記事一覧表示 / 記事閲覧 / カテゴリ分類
- P1: 社員プロフィール閲覧
- P2: 利用規約・プライバシー表示
- P3: アラート/レコメンド

### 通知
- P3: 通知一覧 / レコメンド通知 / Slack連携通知

### 記事
- P0: 記事作成・公開・編集・削除 / 閲覧率表示 / カテゴリ分類
- P1: 記事へのタグ追加 / ユーザー別の分析表示
- P2: 関連記事表示 / おすすめ記事表示 / いいね / コメント
- P3: 温度表示

### エンゲージメント
- P0: 各記事に対するユーザーの既読・未読
- P1: ユーザー一覧・検索
- P2: ユーザー興味の可視化
- P3: 部署別の既読/未読、興味可視化

### システム管理
- P0: 権限カスタム（閲覧/編集/削除） / パスワードリセット
- P1: ログインシステム管理ページ
- P2: 有料オプションON/OFF
- P3: できることの制限設定

### 追加機能の検討
- P2: 温度スコア
- P3: 興味マッチング / CSV出力 / 限定公開記事 / 温度の多面表示

### 分析内容（データ）
- P0: 未読/既読 / ユーザーの閲覧判定
- P1: 離脱率
- P2: 温度（興味）
- P3: レコメンドに必要なデータ / AI機能
